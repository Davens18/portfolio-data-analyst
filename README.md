# Data Analyst Portfolio Dashboard 📊

Portafolio interactivo que demuestra la capacidad de integrar APIs de terceros con visualizaciones dinámicas.

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 (Grid & Flexbox), JavaScript Vanilla.
- **Visualización:** Chart.js para el análisis de métricas musicales.
- **API:** Spotify Web API.
- **Deployment:** GitHub Pages.

## 📈 Insights del Dashboard
El gráfico de barras analiza métricas de **Bailabilidad** obtenidas de mis artistas favoritos (Grupo Niche, Willie Colón, etc.), demostrando el procesamiento de objetos JSON y su conversión a interfaces visuales.

## 🔒 Nota de Seguridad
Este proyecto utiliza un flujo de datos asíncrono. Por seguridad, las credenciales de la API no están expuestas en el código cliente.
